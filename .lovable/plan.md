
# Скрыть черновик ИИ для отвеченных отзывов

## Проблема

На карточках отзывов, на которые уже дан ответ (любой -- автоответ или ручной), продолжает отображаться секция "Черновик ИИ" с кнопками действий. Черновик нужен только для отзывов в статусе "pending" (ожидают ответа).

## Решение

Одна строка в файле `src/components/ReviewCard.tsx`:

**Было:**
```typescript
const hasDraft = aiDraft && status !== "sent" && !isArchived;
```

**Стало:**
```typescript
const hasDraft = aiDraft && status === "pending";
```

Черновик ИИ будет виден только для отзывов со статусом `pending`. Для всех остальных (`auto`, `sent`, архивных) -- скрыт.
