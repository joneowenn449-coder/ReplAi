

# Отключить верификацию email при регистрации

## Что сделать

Включить автоподтверждение email в настройках аутентификации, чтобы пользователи могли сразу войти после регистрации без подтверждения почты.

## Техническая реализация

### 1. Настройка аутентификации
Использовать инструмент `configure-auth` для включения параметра `autoconfirm` email-регистраций. Это единственное изменение на стороне бэкенда.

### 2. Упрощение страницы Auth.tsx
Убрать сообщение "Проверьте почту для подтверждения" после регистрации -- пользователь будет автоматически залогинен и перенаправлен на главную страницу.

## Затрагиваемые файлы
- Настройки аутентификации (configure-auth)
- `src/pages/Auth.tsx` -- убрать уведомление о необходимости подтверждения email
