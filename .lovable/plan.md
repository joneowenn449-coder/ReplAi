
# Убрать названия товаров из выпадающего списка артикулов

## Что меняется

В компоненте `RecommendationsSection.tsx` в выпадающем списке выбора артикула сейчас отображается формат:

```
471329594 — Сумка большая эко замшевая шоппер на плечо
```

Нужно оставить только артикул:

```
471329594
```

## Изменение

### Файл: `src/components/RecommendationsSection.tsx`

Строка 86 -- убрать `— {a.name}` из текста `SelectItem`:

- Было: `{a.article} — {a.name}`
- Станет: `{a.article}`
