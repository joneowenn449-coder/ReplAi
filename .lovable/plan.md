
# Добавить логотип и название "ReplAi" в шапку

## Что меняется

В самом верху шапки (над навигационными табами) появится строка с логотипом и названием сервиса "ReplAi".

## Дизайн

```text
+----------------------------------------------------------+
| [logo] ReplAi                                            |
|                                                          |
| [Отзывы] [Чаты] [AI]                                    |
|                                                          |
| Отзывы Wildberries                        [Настройки]    |
| Управление отзывами и автоответами с ИИ                  |
+----------------------------------------------------------+
```

Логотип -- минималистичная SVG-иконка в стиле Apple (скруглённый прямоугольник с буквой "R" внутри или абстрактный символ диалога/ответа). Цвет -- фирменный фиолетовый (`primary`). Название "ReplAi" отображается жирным шрифтом, часть "Ai" выделена фиолетовым цветом для акцента на ИИ-составляющую.

## Изменение

### Файл: `src/components/Header.tsx`

1. Добавить блок с логотипом и названием перед `NavTabs` (строка 16):
   - SVG-логотип: скруглённый прямоугольник (как иконка приложения в стиле Apple) с символом чата/ответа внутри, залитый `hsl(var(--primary))`
   - Текст "Repl" обычным цветом `text-foreground`, "Ai" -- цветом `text-primary`
   - Размер логотипа: 32x32px
   - Шрифт названия: `text-xl font-bold tracking-tight`

2. Обернуть логотип + название в `div` с `flex items-center gap-2 mb-4`

Никакие другие файлы не затрагиваются.
