
# Заменить поле ввода артикула на выпадающий список

## Что меняется

Сейчас для добавления рекомендации используется текстовое поле `Input` для ввода артикула вручную. Нужно заменить его на `Select` с выпадающим списком всех артикулов из базы, исключая текущий выбранный артикул (source) и уже добавленные рекомендации.

## Изменение

### Файл: `src/components/RecommendationsSection.tsx`

1. **Заменить `useState` для `newArticle`**: тип остается `string`, но теперь значение выбирается из списка, а не вводится вручную.

2. **Заменить `Input` для артикула (строки 135-140)** на `Select`:
   - Список артикулов берётся из того же `articles`, что и верхний селектор
   - Фильтрация: исключается `selectedArticle` (текущий артикул-источник) и все артикулы, уже добавленные в `recommendations`
   - Формат отображения: только номер артикула (как в верхнем списке)

3. **Убрать поле "Название"** (`Input` для `newName`, строки 141-146): название будет подтягиваться автоматически из списка `articles` при выборе артикула в `handleAdd`.

4. **Обновить `handleAdd`**: при добавлении искать название товара из массива `articles` по выбранному артикулу, передавать его как `targetName`.

5. **Сбрасывать `newArticle`** в `""` после успешного добавления (уже реализовано).

### Результат UI

```
Было:
[_введите артикул_] [_Название (необяз.)_] [+]

Станет:
[Выберите артикул... ▼] [+]
```

В выпадающем списке не будет текущего артикула и уже добавленных рекомендаций.
