

# Централизованное меню профиля

## Проблема

Сейчас верхняя панель перегружена отдельными элементами:
- Баланс токенов (плашка)
- Ссылка "Админ" (кнопка)
- Email пользователя (текст)
- Кнопка выхода (иконка)
- Кнопка "Настройки" (отдельно внизу заголовка)

Всё это разбросано и выглядит неаккуратно, особенно на мобильных экранах.

## Решение

Заменить все эти элементы одним компактным **выпадающим меню профиля** (DropdownMenu). В шапке останется только:
- Логотип ReplAi (слева)
- Навигационные табы (по центру или слева)
- Аватар/кнопка профиля (справа) -- при клике открывается меню

## Структура выпадающего меню

```text
+----------------------------------+
|  user@email.com                  |
|  (мелким текстом под email)      |
+----------------------------------+
|  Coins  123 токенов              |
+----------------------------------+
|  Settings  Настройки             |
|  Shield    Админ-панель  (*)     |
+----------------------------------+
|  LogOut    Выйти                 |
+----------------------------------+

(*) -- только для администраторов
```

## Что изменится визуально

- Шапка станет чище: лого слева, табы, аватар справа
- Заголовок "Отзывы Wildberries" и подзаголовок останутся, но без кнопки "Настройки" рядом (она переедет в меню профиля)
- На мобильных вместо скрытых элементов -- одна понятная кнопка с аватаром

## Технические детали

### Изменяемые файлы

1. **`src/components/Header.tsx`**
   - Убрать из правой части шапки: плашку токенов, кнопку "Админ", email, кнопку выхода
   - Добавить компонент `DropdownMenu` (из shadcn/ui, уже установлен) с триггером -- кнопка-аватар (первая буква email или иконка User)
   - Внутри DropdownMenu:
     - `DropdownMenuLabel` -- email пользователя
     - `DropdownMenuSeparator`
     - Пункт с балансом токенов (не кликабельный, просто отображение)
     - `DropdownMenuSeparator`
     - Пункт "Настройки" -- вызывает `onSettingsClick`
     - Пункт "Админ-панель" (условно, только для isAdmin) -- переход на /admin
     - `DropdownMenuSeparator`
     - Пункт "Выйти" -- вызывает `signOut`
   - Убрать кнопку "Настройки" из нижней части шапки (блок с заголовком)

2. **Никаких новых файлов не нужно** -- DropdownMenu уже есть в `src/components/ui/dropdown-menu.tsx`

### Примерная структура нового Header

```text
+-------------------------------------------------------+
|  [Logo ReplAi]                          [Avatar ▼]    |
+-------------------------------------------------------+
|  [Отзывы] [Чаты (3)] [AI аналитик]                    |
+-------------------------------------------------------+
|  Отзывы Wildberries                                    |
|  Управление отзывами и автоответами с помощью ИИ       |
+-------------------------------------------------------+
```

Кнопка "Настройки" убирается из блока заголовка -- теперь она в меню профиля. Заголовок остаётся как информационный блок без действий.
