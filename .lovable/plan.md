

# Добавить шильдик "Автоответ" / "Ручной" к отправленному ответу

## Суть

Сейчас у отправленного ответа просто написано "Отправленный ответ:" без различия, как он был отправлен. Нужно добавить маленький бейдж-шильдик рядом с этой надписью:

- Статус `auto` --> шильдик **"Автоответ"** (фиолетовый)
- Статус `sent` --> шильдик **"Ручной"** (зелёный)

## Что будет изменено

**Файл:** `src/components/ReviewCard.tsx`

В секции отправленного ответа (строка 212-217) добавить бейдж рядом с текстом "Отправленный ответ:":

```
Отправленный ответ:  [Автоответ]   -- или --   [Ручной]
```

Бейдж будет реализован через компонент `Badge` из `src/components/ui/badge.tsx`:
- "Автоответ" -- фиолетовый фон (`bg-purple-100 text-purple-700`)
- "Ручной" -- зелёный фон (`bg-green-100 text-green-700`)

Так же для архивных отзывов, где тоже показывается ответ, шильдик будет определяться по тому, был ли исходный статус `auto` или `sent`. Поскольку архивные отзывы сейчас не сохраняют исходный способ отправки, шильдик будет показываться только для статусов `auto` и `sent`.

## Затрагиваемый файл

| Файл | Изменение |
|------|-----------|
| `src/components/ReviewCard.tsx` | Добавить бейдж "Автоответ" или "Ручной" в секцию отправленного ответа |

## Результат

У каждого отвеченного отзыва будет чётко видно, как он был обработан -- автоматически или вручную.
