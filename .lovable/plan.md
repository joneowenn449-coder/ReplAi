

# Удаление неиспользуемого статуса "Новые"

## Проблема

Статус `"new"` нигде не присваивается при синхронизации отзывов. Логика работы такая:
- Режим "ручной" для рейтинга --> отзыв сразу получает статус `"pending"` (ожидает)
- Режим "авто" для рейтинга --> отзыв отправляется и получает статус `"auto"` (автоответ)

Таб "Новые" всегда показывает 0 и только занимает место в интерфейсе.

## Что будет сделано

### 1. Убрать таб "Новые" из фильтров

**Файл:** `src/components/FilterTabs.tsx`

Удалить элемент `{ id: "new", label: "Новые", count: counts.new }` из массива фильтров.

### 2. Убрать карточку "Новые" из статистики

**Файл:** `src/components/StatsCards.tsx`

Удалить карточку с `newCount` -- останутся только: Ожидают, Автоответ, Отправлено.

### 3. Убрать подсчёт "new" из Index.tsx

**Файл:** `src/pages/Index.tsx`

Убрать `stats.new` и `counts.new` из вычислений, а также проп `newCount` из StatsCards.

### 4. Убрать "new" из типа статуса

**Файл:** `src/hooks/useReviews.ts`

Изменить тип статуса с `"new" | "pending" | "auto" | "sent" | "archived"` на `"pending" | "auto" | "sent" | "archived"`.

## Затрагиваемые файлы

| Файл | Изменение |
|------|-----------|
| `src/components/FilterTabs.tsx` | Удалить таб "Новые" |
| `src/components/StatsCards.tsx` | Удалить карточку "Новые" |
| `src/pages/Index.tsx` | Убрать подсчёт и проп newCount |
| `src/hooks/useReviews.ts` | Убрать "new" из типа статуса |

## Результат

Интерфейс станет чище -- только реально работающие статусы: Ожидают, Автоответ, Отправлено, Архив.
