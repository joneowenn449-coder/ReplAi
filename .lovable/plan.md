
# Исключение рекомендаций для отзывов с низким рейтингом (1-3 звезды)

## Суть изменения

В ответ на отзывы с рейтингом 1, 2 или 3 звезды ИИ больше не будет предлагать другие товары бренда. Это поможет избежать неуместных предложений в ситуациях, когда покупатель недоволен качеством или опытом использования.

## Что изменится

**Файл**: `supabase/functions/generate-reply/index.ts`

В блоке формирования инструкций по рекомендациям (строки 86-93) добавится проверка рейтинга. Инструкция `recommendationInstruction` будет формироваться только в том случае, если рейтинг отзыва 4 или 5 звезд.

```typescript
// Было:
if (recommendations && recommendations.length > 0) { ... }

// Станет:
if (recommendations && recommendations.length > 0 && review.rating >= 4) { ... }
```

## Технические детали

- Изменяется 1 файл: `supabase/functions/generate-reply/index.ts`
- В логи консоли будет добавлено сообщение, если рекомендации пропускаются из-за низкого рейтинга, для удобства отладки.
- Остальная логика (выбор моделей, анализ фото, определение отказа) остается без изменений.
