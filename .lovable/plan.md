
# Исправление видимости меню трёх точек в сайдбаре

## Проблема

Кнопка с тремя точками (меню действий) у каждого чата в сайдбаре скрыта по умолчанию и появляется только при наведении курсора (`opacity-0 group-hover:opacity-100`). Это неочевидно для пользователей.

## Решение

Сделать кнопку трёх точек всегда видимой — убрать `opacity-0 group-hover:opacity-100`, оставить постоянную видимость с мягкой прозрачностью (`opacity-50 hover:opacity-100`), чтобы она не отвлекала, но всегда была заметна.

## Технические детали

**Файл:** `src/components/AiChatSidebar.tsx`, строка 117

Изменение класса кнопки:
- Было: `opacity-0 group-hover:opacity-100 p-0.5 rounded-md hover:bg-secondary transition-opacity`
- Станет: `opacity-40 hover:opacity-100 p-0.5 rounded-md hover:bg-secondary transition-opacity`

Одно минимальное изменение — три точки будут видны всегда.
