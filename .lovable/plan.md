
# Добавить время к дате отзыва

## Что нужно сделать

Сейчас на карточке отзыва отображается только дата: **"5 фев 2025"**. Нужно добавить время: **"5 фев 2025 в 14:32"**.

## Изменение

### Файл: `src/pages/Index.tsx`

Строка 105 -- изменить формат даты с:

```
"d MMM yyyy"
```

на:

```
"d MMM yyyy 'в' HH:mm"
```

Это единственное изменение. Формат `HH:mm` добавит часы и минуты (24-часовой формат), а предлог "в" визуально разделит дату и время.

Результат: **"5 фев 2025 в 14:32"** -- такой же формат уже используется для отображения времени последней синхронизации в этом же файле (строка 56).
